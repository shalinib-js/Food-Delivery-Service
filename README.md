# Food Delivery Service

A modern, responsive React app for browsing, ordering, and managing food deliveries.
Built with beginner-friendly React patterns, styled with Bootstrap and custom CSS, and deployed via Netlify.

## Features

- Browse food items with responsive cards.
- Apply coupon codes for discounts
- Checkout & Payment section
- Clean, mobile-friendly UI with Bootstrap + CSS
